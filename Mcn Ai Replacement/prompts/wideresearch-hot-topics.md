# WideResearch：全平台热点采集与深度研究提示词

> **用途**：在 Manus / KIMI 多 Agent 架构中执行 WideResearch 任务，并行采集多平台热点与 AI 前沿信息

> **输出**：结构化热点清单 → 导入飞书多维表格

- `商业逻辑`：商业模式、变现路径、创业方法论
- `哲学社会`：AI 与社会、AI 与人类存在、文化批评、前沿思想讨论

在正式搜索前，请先执行以下步骤：

2. **行业时事关键词扩展**：基于本周 AI 行业最新动态（新模型、重要访谈、政策、争议事件），生成 5-10 个时效性关键词，与基础关键词组合用于下游搜索
   - 例：如本周 OpenAI 发布重要公告，则增加相关关键词
   - 例：如「2028 智能危机」话题在 X 上发酵，则在国内平台同步搜索对应词
3. **跨语言映射**：将海外热词映射到国内对应表达，确保双向覆盖
4. 请在输出末尾列出本次实际使用的动态扩展关键词，供下次优化参考

## 使用说明

1. 将上述提示词复制到 Manus 或 KIMI 多 Agent 任务中执行
2. 执行前可根据本周 AI 重大事件手动补充到「关键词动态生成策略」的背景说明
3. 执行后审核结果，将有效热点录入飞书多维表格（按 [hot-topics-schema](../knowledge-base/schemas/hot-topics-schema.md) 格式）
4. 将「AI 前沿人物动态摘要」汇总到知识库，作为哲学/社会视角内容素材
5. 保存「本次搜索元反馈」，每月整合一次用于提示词迭代
