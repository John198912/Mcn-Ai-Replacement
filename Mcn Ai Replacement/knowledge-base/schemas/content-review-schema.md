# 飞书多维表格Schema：内容数据表

## 表名：📊 内容数据复盘

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| 内容标题 | 文本 | 发布时的标题 | "35岁被裁后..." |
| 平台 | 单选 | 抖音/小红书/B站/视频号 | 抖音 |
| 发布日期 | 日期 | - | 2026-03-20 |
| 内容形式 | 单选 | 口播/图文/长文/教程/直播 | 口播视频 |
| 时长(秒) | 数字 | 视频时长 | 300 |
| 选题来源 | 单选 | 热点蹭/对标启发/自发/商单 | 热点蹭 |
| 关联热点 | 关联 | 链接到热点跟踪表 | - |
| --- | --- | **公开数据** | --- |
| 播放量 | 数字 | - | 50000 |
| 点赞数 | 数字 | - | 2000 |
| 评论数 | 数字 | - | 150 |
| 收藏数 | 数字 | - | 500 |
| 转发数 | 数字 | - | 100 |
| 新增粉丝 | 数字 | 该条内容带来的 | 200 |
| --- | --- | **后台数据（手动填）** | --- |
| 完播率(%) | 数字 | 后台导出 | 35 |
| 5秒留存率(%) | 数字 | 后台导出 | 65 |
| 流量来源-推荐(%) | 数字 | - | 80 |
| 流量来源-搜索(%) | 数字 | - | 10 |
| 流量来源-关注(%) | 数字 | - | 8 |
| 流量来源-其他(%) | 数字 | - | 2 |
| --- | --- | **分析字段** | --- |
| 互动率(%) | 公式 | (点赞+评论+收藏)/播放量*100 | 5.3 |
| 转粉率(%) | 公式 | 新增粉丝/播放量*100 | 0.4 |
| 收藏点赞比 | 公式 | 收藏/点赞 | 0.25 |
| 内容评级 | 单选 | 爆款/良好/一般/低于预期 | 良好 |
| 问题诊断 | 长文本 | content-reviewer Skill产出 | "5秒留存偏低→开头过长" |
| 优化建议 | 长文本 | 下次改进方向 | "缩短开头到3秒内" |
| 备注 | 文本 | 补充信息 | - |
