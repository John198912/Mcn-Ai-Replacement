# title-optimizer Skill

## 概述

基于脚本/内容摘要和平台特性，生成多版标题候选，基于对标爆款标题模式预估CTR，并给出封面构图建议。

## 触发场景

- 脚本完成后
- 创作者请求"帮我优化标题"

## 输入

1. **脚本摘要**：核心内容+关键信息点
2. **平台**：抖音/小红书/B站
3. **参考标题模式**（可选）：对标爆款的标题分析

## 标题生成策略

| 策略 | 适用场景 | 示例模式 |
|------|---------|---------|
| 反常识型 | 挑战认知的观点 | "你以为的XX其实是在YY" |
| 数字型 | 方法论/清单 | "3个让我XX的YY" |
| 痛点直击 | 受众焦虑点 | "35岁被裁后，我终于明白了..." |
| 悬念型 | 故事/经历 | "做了X之后，发生了出乎意料的事" |
| 对比型 | 认知差 | "XX和YY的人，差距在这里" |
| 身份认同 | 受众自我标签 | "每个想XX的人，都需要知道..." |

## 输出

```markdown
### 标题优化建议

**内容**：[脚本摘要]
**平台**：[目标平台]

#### 候选标题（按预估CTR排序）

| # | 标题 | 策略 | 预估CTR | 搜索词匹配 | 说明 |
|---|------|------|---------|-----------|------|
| 1 | ... | 反常识 | ⭐⭐⭐⭐⭐ | 高 | ... |
| 2 | ... | 痛点 | ⭐⭐⭐⭐ | 中 | ... |
| ... | | | | | |

#### 封面建议
- 构图方式：人物出镜/文字卡/场景图
- 配色建议
- 关键文字（如果是文字封面）
- 参考对标：[对标创作者的类似封面风格]
```
