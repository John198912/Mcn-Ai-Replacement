# viral-content-analyzer Skill

## 概述

对对标创作者的高互动内容进行结构化拆解，提取hook手法、内容节奏、情绪设计等可迁移元素，反哺选题和脚本创作。

## 触发场景

- 创作者指定某条爆款内容请求拆解
- M19对标追踪发现新爆款后自动触发

## 输入

1. **爆款内容信息**：标题、平台、数据（点赞/评论/收藏）、内容摘要或脚本文本
2. **创作者画像**：该爆款所属创作者的档案

## 分析维度

```markdown
### 爆款拆解报告

#### 基本信息
- 创作者/平台/发布时间/数据表现

#### 标题分析
- Hook类型：悬念型/数字型/反常识型/痛点直击型/对比型
- 关键词策略：SEO友好度/搜索词匹配
- 情绪触发点

#### 内容结构拆解
- **前3秒**：如何留人（具体手法）
- **起承转合**：信息节奏/密度变化曲线
- **信息增量分布**：每个段落的核心增量点
- **情绪曲线**：tension/release节奏
- **结尾设计**：CTA方式/互动引导

#### 数据解读
- 点赞/评论比例 → 内容引发的情绪类型
- 收藏/点赞比例 → 实用价值信号
- 转发/点赞比例 → 社交货币价值

#### 可迁移元素
1. [可直接用在我内容中的手法]
2. [需要适配调整的手法]
3. [不适合我人设的手法]
```

## 依赖

- 对标创作者档案（`creator-profiler`产出）
- 创作者个人Profile（用于"可迁移"判断）
